# 🏦 Banking System in C++

A console-based Banking System implemented using **Object-Oriented Programming (OOP)** concepts in C++. This project simulates basic banking operations such as account creation, deposits, withdrawals, and balance inquiry.

---

## 🛠️ Tech Stack

- Language: **C++**
- Concepts Used:
  - Classes & Objects
  - Encapsulation
  - File Handling
  - Conditional Statements
  - Loops
